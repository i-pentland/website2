Concept

The app solves a small real-life problem: deciding whether now is a good time to step outside for a break. It turns weather data into a simple answer.

APIs Used

  • Geolocation API for the user’s current location
  • Open-Meteo Geocoding API for city search
  • Open-Meteo Forecast API for weather data
  • localStorage to save the last searched city
